# cache_hash

Crystal implementation of https://github.com/DougEverly/CacheHash



## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cache_hash:
       github: DougEverly/cache_hash
   ```

2. Run `shards install`
